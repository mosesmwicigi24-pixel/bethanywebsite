# Legacy storefront snapshot

`legacy-bethanyhouse-2026-07-19.tar.gz` — wget mirror of the CodeIgniter
storefront as served at https://bethanyhouse.co.ke on 19 July 2026
(homepage + /shop with page requisites, 159 files), taken immediately
before the cutover to the Next.js storefront in `storefront/`.

The full legacy application (POS + back office, still in production
for staff) lives in this repository's root (`application/`, `assets/`)
and keeps running behind the same domain on its own paths.

To browse the snapshot: `tar xzf legacy-bethanyhouse-2026-07-19.tar.gz`
and open `legacy-snap/bethanyhouse.co.ke/index.html`.
